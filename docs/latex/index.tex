It\textquotesingle{}s also a publisher, due to the function head\+\_\+control in order to move the robot head. And moreover it\textquotesingle{}s a client of the the action server according to the robot movement. In this way, by checking several condition, it\textquotesingle{}s possible to resarch the desired state inside the FSM. ~\newline
