\doxysubsection{Detailed Description}
~\newline
 

Module of the node \mbox{\hyperlink{namespace_ball_detection}{Ball\+Detection}}. After a pre-\/processing phase, due to a simple algorithm it recognizes the mean color of the ball detected. This is done by computing the distance between the color values stored in the dictionary and the mean of the circle image. The color with the smallest distance is returned. Moreover the method {\ttfamily generate\+Boundaries} returns to the \mbox{\hyperlink{namespace_ball_detection}{Ball\+Detection}} node the lower and the upper mask for the ball found.
