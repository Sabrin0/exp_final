Detailed Description

This class is the core of the user node.

It handles the communication between the command manager and the user interface.

◆ backUser()

def userPlay.console_manager.backUser (   self)

Methods that saves the input command play in order to call the robot to the user.

This is done by sending the string play over the topic currentState when the command manager recives the command, it sends the user position as Nav Stack goal. It also checks if the robot is in the normal state, otherwise it just ignore the input.

Attributes

self.msg_play: ros message It contains:

  • play: type Bool, it checks if the user inputs the correct command
  • color: type String, it stores the color selected by the user. Initialized as empty string

command: String, It stores the raw_input() data from the user.
